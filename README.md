# Welcome to againstU

## What's againstU?

**againstU** is a Website that allows to know your stats and stats of your opponents on **Dota** and **League of Legends** games.
Our website use the *Dota* and *League Of Legend* **API**

## Technologies Used

* NodeJS
* ExpressJS
* MomentJS
* EJS
* Axios
